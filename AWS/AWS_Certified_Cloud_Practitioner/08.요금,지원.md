## (1) **AWS 프리 티어**   
> 지정된 기간 동안 특정 서비스 무료 이용

- 상시 무료

    * Amazon DynamoDB 매월 25GB 무료 스토리지

- 12개월 무료

- 평가판

## (2) **AWS 요금 개념**

### 1. **요금 적용 방식**

- 실제 사용한 만큼만 지불

- 예약하는 경우 비용 감소 (최대 72%)

- 많이 사용할수록 볼륨 기반 할인으로 비용 감소
    * ex) S3 스토리지 공간을 많이 사용할수록 GB당 비용 감소

### 2. **AWS 요금 계산기**   
> AWS 서비스를 탐색하고 사용 사례에 대한 비용 추정 가능

- - -

## (3) **결제 대시보드**
> AWS 청구서를 결제, 사용량 모니터링, 비용 분석 및 제어

- 이번 달 누계 금액을 지난 달과 비교하고, 현재 사용량 기준으로 내달 사용량을 예측

- 서비스별 월 누계 지출 확인

- 서비스별 프리 티어 사용량 확인

- Cost Explorer에 액세스하여 예산 생성

- Saving Plans를 구매하고 관리

- AWS 비용 및 사용 보고서를 게시

- - -

## (4) **통합 결제**

> **AWS Organizations** 는 통합 결제 옵션을 제공

- 조직의 모든 AWS 계정에 대한 단일 청구서를 받을 수 있음

- 기본적으로 조직에 허용되는 최대 계정 수는 4개, but AWS Support에 문의하여 할당량을 늘릴 수 있음

- 월별 청구서에서 각 계정에서 발생한 항목별 요금 검토 

- 조직의 계정 전체에서 대량 할인 요금, Savings Plans, 예약 인스턴스 공유 가능

- - -

## (5) **AWS 예산 (Budget)**
> 예산을 생성하여 서비스 사용, 서비스 비용 및 인스턴스 예약을 계획

- 정보가 하루에 세 번 업데이트됨 => 한도 측정에 용이

- 사용량이 예산 금액을 초과하거나 예상되면 알려주는 **사용자 지정 알림** 설정
    * ex) 예산에서 사용량이 지정 금액이 절반에 도달했을 때 알림 설정 가능

- - -

- 사용자 지정 필터 및 그룹을 적용하여 데이터 분석

## (7) **AWS Support Plan**   
> 4가지 Support Plans

### 1. **Basic Support**   
> 모든 AWS 고객에게 무료 제공

- 백서, 설명서 및 지원 커뮤니티에 대한 액세스 포함

- **AWS Trusted Advisor** 검사에 액세스 가능

- **AWS Personal Health Dashboard** 사용 가능
    > 이벤트 발생할 때 알림 및 수정 지침을 제공


### 2. **Developer Support**

- 모범 사례 지침

- 클라이언트 측 진단 도구

- AWS 제품, 기능 및 서비스를 함께 사용하는 방법에 대한 지침으로 구성된 **빌딩 블록 아키텍처** 지원

### 3. **Business Support**
> 이전 plan에 더하여 추가 기능

- 특정 요구 사항을 가장 잘 지원할 수 있는 AWS 제품, 기능 및 서비스를 식별하기 위한 사용 사례 지침

- 모든 **AWS Trusted Advisor** 검사

- 운영 체제 및 애플리케이션 스택 구성 요소와 같은 타사 소프트웨어에 대한 제한된 지원


### 4. **Enterprise Support**   
> Basic, Developer, Business Support Plan에 포함된 기능 외에도 추가 기능

- 회사의 특정 사용 사례 및 애플리케이션을 지원하기 위한 컨설팅 관계인 **애플리케이션 아키텍처 지침**

- **인프라 이벤트 관리 지원** : 회사가 사용 사례를 더 잘 이해할 수 있도록 돕는 AWS Support와의 단기 계약. 또한, 회사에 **아키텍처** 및 **확장 지침** 제공

- **기술 계정 관리자 (TAM)**

### **기술 계정 관리자 (TAM)**   
> 회사 측에서 TAM이 AWS 측 주 연락 창구

- 애플리케이션 계획, 배포, 최적화할 때 TAM이 **지속적 커뮤니케이션** 하면서 권장 사항, 아키텍처 검토를 제공

- 모든 AWS 서비스에 대한 전문 지식 제공
