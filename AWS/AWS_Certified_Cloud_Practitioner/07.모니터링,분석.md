모니터링 및 분석
===============

## (1) **Amazon CloudWatch**   
> 다양한 지표를 모니터링 및 관리, 해당 지표의 데이터를 기반으로 **Alarm** 작업을 구성하는 웹 서비스

- **지표** 를 사용하여 리소스의 데이터 포인트를 나타냄

- 지표를 CloudWatch로 전송하여 시간 경과에 따라 성능의 변화를 보여주는 그래프를 자동 생성

## (2) **CloudWatch Alarm**   
> 지표의 값이 미리 정의된 임계값을 상회 or 하회할 경우 자동으로 작업 수행
