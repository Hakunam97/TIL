## (2) AZ(Availability Zone)
- 가용 영역은 리전 내의 단일 데이터 센터 또는 데이터 센터 그룹   

- AWS는 AZ간에 높은 처리량과 지연시간(콘텐츠가 요청된 시점과 수신된 시점 간의 차이)이 짧은 네트워킹 제공   

- 재해가 발생할 경우 여러 가용 영역이 영향 받지 않도록 수십 마일 멀리 떨어져 있음
