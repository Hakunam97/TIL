### range

* 파라미터 2개 range
```python
for i in range(3, 11):  # 3 ~ 10
    print(i)
```

* 파라미터 1개 range
```python
for i in range(10): # 0 ~ 9
    print(i)
```

* 파라미터 3개 range
```python
for in range(3, 17, 3): # 3 ~ 16 / 마지막 3씩 간격 => [3, 6, 9, 12, 15]
    print(i)
```
