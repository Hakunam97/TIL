TCP/IP 4 Layer
===

TCP/IP - Application Layer (응용계층)
---

* Application으로 제공되는 서비스는 표현 계층과 세션 계층에서 정의

* 네트워크의 packet 전송을 제어, 데이터 전송 경로를 선택
